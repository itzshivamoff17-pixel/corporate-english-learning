Compliance Terms

Following the rules.

GDPR

Meaning: General Data Protection Regulation.

Example: "We must be GDPR compliant."

Audit

Meaning: Official inspection.

Example: "We passed the security audit."

Regulation

Meaning: Rule or law.

Example: "Banking regulations are strict."

Privacy Policy

Meaning: Data handling rules.

Example: "Update the privacy policy on the site."

Consent

Meaning: Permission.

Example: "User consent is required for cookies."

Data breach

Meaning: Leaked info.

Example: "Report the data breach immediately."

Risk assessment

Meaning: Evaluating danger.

Example: "Conduct a risk assessment."

Whistleblower

Meaning: Person revealing wrongdoing.

Example: "Whistleblowers are protected by law."

Code of Conduct

Meaning: Behavior rules.

Example: "Violating the Code of Conduct can lead to firing."

Governance

Meaning: Management framework.

Example: "Data governance is a priority."
