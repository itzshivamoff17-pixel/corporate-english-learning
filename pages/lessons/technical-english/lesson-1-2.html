Data Flow Vocabulary

Moving beyond "The data goes here," a professional engineer uses precise verbs to describe the lifecycle of data within a system.

Precise Technical Verbs

Ingest
The initial entry of data from external sources. "Our Kafka layer ingests events from the UI clients."
Propagate
The spread of data/state across nodes or services. "Changes to the config are propagated to all regions within 5 seconds."
Aggregate
Combining multiple data streams into one. "The analytics engine aggregates raw logs into hourly metrics."
Stale
Describing outdated data. "Using a TTL of 1 second prevents the cache from serving stale data."

Pro Pattern: Event-Driven Phrasing

"When a user [triggers X], the system [emits an event], which is then [consumed by Y] to [perform Z]."
